## Project Propose
___
- config static to dynamic site with django framework
- ```CRUD``` operation
- sign in & sign up
- add to cart
- track order status & view order detail
- admin dashboard and client side

## Feature
___
- client
  - Shopping, filter by brand, category and search any product.
  - Sign Up & Sign In to shopping (add to cart, view order and checkout)
  - add product to card, update and delete cart item.
  - process checkout, view order detail and order status. (no payment required.)
- admin dashboard
  - manage brand,category,slider and product.
  - analyze dashboard to view sale earning, new user...
  
## Technologies:
___
- Frontend
  - Bootstrap
  - Jinja2
  - JQuery (some path of website like: add to cart, process checkout...)
- Database
  - MongoDB with ```mongoengine version 0.29.1```
- Backend
  - Django Framework: ```version 5.2.6```
- Deployment: ```Docker``` with ```AWS EC2```
